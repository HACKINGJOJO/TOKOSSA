RESTE A FAIRE — Classe par priorite
------------------------------------------------------------

=== PRIORITE HAUTE ===

[H1] Programme de fidelite
     - Le modele LoyaltyPoints existe deja dans le schema Prisma
     - Points gagnes par achat (ex: 1 point / 100 FCFA)
     - Affichage des points sur la page profil
     - Conversion points -> reduction au checkout
     - Historique des points (gains + depenses)

[H2] Systeme de parrainage
     - Code unique par client (genere apres 1er achat)
     - Partage via WhatsApp (lien avec code)
     - Le parrain gagne X FCFA quand son filleul achete
     - Tableau de bord parrainage sur la page profil
     - Tres viral dans le contexte beninois

[H3] Notifications admin en temps reel
     - Son + badge navigateur a chaque nouvelle commande
     - Notification push navigateur (Web Push API)
     - Compteur commandes en attente sur le dashboard
     - Option email admin a chaque commande

[H4] Gestion des retours
     - Formulaire demande de retour cote client
     - Workflow admin : accepter / refuser / rembourser
     - Timeline de suivi du retour
     - "Retour 7 jours" est affiche partout mais pas implemente

[H5] Page "A propos / Qui sommes-nous"
     - Renforce la confiance (critique pour le e-commerce au Benin)
     - Photos equipe, localisation Cotonou, histoire
     - Valeurs, engagements qualite
     - Liens reseaux sociaux


=== PRIORITE MOYENNE ===

[M1] Mode sombre
     - Le marche mobile africain consomme beaucoup la nuit
     - Reduit fatigue visuelle + usage batterie OLED
     - Toggle dans Navbar ou Footer
     - Sauvegarde preference dans localStorage

[M2] Wishlist / Favoris
     - Bouton coeur sur chaque produit
     - Sauvegarde localStorage (comme recemment consultes)
     - Page "Mes favoris" dans le compte
     - Bouton "Ajouter au panier" depuis la wishlist

[M3] Filtres avances produits
     - Filtre par fourchette de prix (slider min/max)
     - Filtre par note client (3+ etoiles, 4+ etoiles)
     - Filtre "En promotion" / "Nouveau" / "En stock"
     - Filtres combines avec les categories existantes

[M4] Pagination / Infinite scroll
     - 24 produits affiches d'un coup actuellement
     - Pagination classique ou chargement progressif au scroll
     - Necessaire quand le catalogue depasse 50+ produits
     - Parametre ?page=1&limit=12 dans l'API

[M5] Galerie produit amelioree
     - Zoom au tap/hover sur l'image
     - Swipe mobile natif entre images
     - Support video produit (YouTube/Cloudinary embed)
     - Thumbnails cliquables en bas de l'image principale

[M6] Interface admin pour les codes promo
     - CRUD complet des codes promo (creer/modifier/supprimer)
     - Stats d'utilisation par coupon (combien de fois utilise, CA genere)
     - Activation/desactivation rapide
     - Actuellement les promos sont creees uniquement via seed


=== PRIORITE BASSE ===

[B1] PWA (Progressive Web App)
     - Manifest.json + Service Worker
     - "Installer TOKOSSA" comme une app mobile
     - Cache offline du catalogue
     - Notifications push natives
     - Icone sur l'ecran d'accueil

[B2] Multi-langue
     - Francais (defaut) + Fon + Yoruba
     - Differenciateur fort sur le marche beninois
     - i18n avec next-intl ou next-i18next
     - Selecteur de langue dans le Footer

[B3] Export commandes (CSV/PDF)
     - Dashboard admin -> exporter commandes du mois/semaine
     - Factures PDF auto-generees par commande
     - Rapport de ventes avec graphiques
     - Utile pour la comptabilite

[B4] Chat en direct
     - Widget WhatsApp flottant sur toutes les pages
     - Indicateur "En ligne" / "Reponse en 5 min"
     - Alternative : integration Tawk.to ou Crisp (gratuit)
     - Position fixe en bas a droite, ne gene pas le scroll

[B5] Avis photos
     - Permettre aux clients de joindre des photos a leurs avis
     - Upload via Cloudinary
     - Galerie photos clients sur la page produit
     - Augmente enormement la confiance

[B6] Variantes produit
     - Taille, couleur, modele par produit
     - Selection dans ProductDetail
     - Stock par variante
     - Schema Prisma a etendre (ProductVariant)
